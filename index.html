Acerca del proyecto

Esta aplicación de clima fue desarrollada como parte del Módulo 2 de Frontend, implementando tecnologías modernas web como HTML5 semántico, Bootstrap 5 y JavaScript vanilla.

El proyecto muestra información meteorológica de 12 ciudades principales de Latinoamérica, con diseño responsive y navegación intuitiva.
